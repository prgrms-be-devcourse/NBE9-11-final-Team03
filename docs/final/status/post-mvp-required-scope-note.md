# Baton MVP 이후 확장 범위 메모

> 기준일: 2026-07-01
> 주의: 아래 항목은 팀 내부 고도화 우선순위이며 공식 최종 제출 필수 기능이 아니다.

## 우선순위

| 우선순위 | 기능 | 현재 발표 기준 |
| --- | --- | --- |
| P1 | 채팅 | 구현 및 최종 시연 검증 범위만 소개 |
| P1 | 관리자 | 실제 동작 근거가 있을 때만 구현 완료로 소개 |
| P1 | SWAP | PURCHASE 이후 확장 방향으로 소개 |
| P1 | 리뷰/신뢰 점수 | 실제 동작 근거가 있을 때만 구현 완료로 소개 |
| P1 | S3 첨부파일 | 배포 환경 검증 범위만 소개 |
| P2 | 알림 | 후속 로드맵 |
| P2 | 유료 크레딧 충전 | 후속 로드맵 |
| P2 | 추천 고도화 | 후속 로드맵 |

## 발표 원칙

- 공식 필수 제출물과 팀 내부 기능 목표를 구분한다.
- 코드, 테스트, Swagger 또는 UI 시연 근거가 없으면 `확장 계획`으로 표시한다.
- PURCHASE의 완성도를 흐리는 기능 나열보다 검증된 거래 흐름을 우선한다.
