# Baton 회의 문서 안내

회의 안건, 회의 진행 문서, 회의록은 최종 제출 문서와 분리해서 이 폴더에서 관리한다.

## 문서 역할

| 문서 | 역할 |
| --- | --- |
| `meeting-agenda-2026-06-21.md` | 회의 전 논의 안건 |
| `meeting-2026-06-21-mvp-sync.md` | 회의 진행용 문서 |
| `meeting-minutes-2026-06-21.md` | 회의 후 결정 사항 기록 |

## 관리 원칙

- 팀 공유가 필요한 결정 사항은 회의록에 남긴다.
- 최종 제출/발표 문서에 반영할 내용은 `docs/final`의 역할별 문서로 옮긴다.
- 지난 회의 안건은 보존하되, 최신 작업 관리는 `docs/final/status`에서 한다.
